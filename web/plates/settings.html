AHRS Output to EFB: Compatibility Note

This option must be enabled to send AHRS messages to compatible EFBs (WingX, iFly, etc.) over the WiFi interface using the GDL90 protocol. Hardware AHRS or GPS simulated AHRS must also be enabled. This mode is not compatible with ForeFlight, and must be disabled for ForeFlight 7.6 to work at all.

Acknowledge

ForeFlight 'Flight Simulator' AHRS Output: Compatibility Warning

This option enables position, traffic, and AHRS messages over the WiFi interface using the ForeFlight "Flight Simulator" protocol. Hardware AHRS or GPS simulated AHRS must also be enabled. This mode is intended for developer use on the ground only, and may not function correctly while airborne. This mode disables ALL GDL90 protocol traffic, meaning ForeFlight will not show FIS-B weather and NOTAMS, and other applications won't work at all.

Acknowledge
