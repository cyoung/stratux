The Traffic page provides a list of all aircraft position reports received within the last 60 seconds. Each time a new aircraft is reported, it is added to the list. Each time a new report is received for an existing aircraft, the list is updated. If a report for an aircraft has not been received within the last 60 seconds, the aircraft is removed from the list.

For each aircraft, the list includes the following details:

  • Flight is aircraft tail number / call sign (when available) or the ICAO 24-bit code. When the ICAO code is used, it is displayed in gray.
    • 1090 ES traffic is displayed with a light blue background.
    • 978 UAT traffic is displayed with a light tan background.
  • Speed - Reported ground speed, rounded to the nearest 5 knots. Invalid or missing values are shown as '---'.
  • Altitude - Reported pressure altitude, feet MSL. Climb or descent rate is also shown if not in level flight.
  • Course - Reported true course, rounded to the nearest 5°. Invalid or missing values are shown as '---'
  • Location - Reported latitude and longitude, DD° mm' ss''.
  • Age - Age of the last traffic report, seconds.
